# HTMLLint Configuration

## Installation

```sh
npm i @alexlit/config-htmllint -D
```

## Connection

```js
// .linthtmlrc.js
module.exports = {
  ...require('@alexlit/config-htmllint/.linthtmlrc.js'),
};
```
